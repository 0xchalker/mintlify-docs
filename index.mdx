# MEME LABS

MEME LABS là một hệ thống vận hành theo giai đoạn để:

1. Tìm narrative đang nóng.
2. Chọn ra đúng một narrative đáng đánh.
3. Khóa narrative đó thành một concept coin rõ ràng.
4. Biến concept thành content system cho X và media.
5. Launch token trên PumpFun.
6. Lưu lại toàn bộ campaign để học lại và tái sử dụng.
7. Vận hành public loop và đưa bài học ngược về hệ thống.

Tài liệu này không dùng để liệt kê toàn bộ skill nội bộ. Nó dùng để giúp anh review:

- hệ thống đang chạy vì mục đích gì
- từng giai đoạn cần làm gì
- input nào đi vào từng giai đoạn
- output nào phải đi ra
- file nào là file quyết định
- khi nào được qua bước tiếp theo
