# Architecture

## Four layers

### 1. Filesystem truth

- `.narrative/`
- `.concepts/`
- `.campaigns/`
- `.agents/evals/`

These folders are canonical.

### 2. Control plane

- `.agents/skills/`
- `.agents/workflows/memelabs/`

This layer defines stage ownership, contracts, gates, and routing.

### 3. Runtime

- `scripts/`
- `prompts/`
- `output/`

This layer executes scouting, review, posting, sync, and helper tasks.

### 4. Control Room mirror

- `output/control-room/state.json`
- `public/control-room-state.json`
- `src/routes/ops*`

This layer visualizes the system without replacing the canonical artifacts.

## Why this matters

The Control Room gives new operators and new AIs a real operating surface.

Without it, MEME LABS is easy to treat like a folder maze.
