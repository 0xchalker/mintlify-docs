# Stage 1 Orchestration

## Internal flow

1. prepare Grok
2. run raw discovery lanes
3. parse raw reports
4. freeze review scope
5. review evidence and assets
6. write batch audit
7. write Trend Final
8. build trend package
9. refresh the Control Room mirror

## Public trust flow

When public logging is enabled:

1. root run post
2. raw narrative replies
3. narrative summary subthreads
4. CEO replies
5. evidence replies
6. asset replies
7. batch audit reply
8. separate Trend Final thread

## Transport rule

- browser X is the default Stage 1 public transport
- X API is optional
- internal artifacts remain canonical
