# Workspace Map

## Root folders

### `/.agents/`

Control plane của hệ thống.

### `/.narrative/`

Nơi lưu các narrative batch đã được quét và phân tích.

### `/.concepts/`

Nơi lưu concept package của mỗi ticker đã được khóa.

### `/.campaigns/`

Nơi lưu kết quả cuối cùng của một campaign.

### `/prompts/`

Prompt sources cho các lane runtime.

### `/scripts/`

Scripts chạy browser automation và các helper chính của hệ thống.

### `/docs/`

Nguồn trang Mintlify để review workflow và output contracts.

### `/output/` (tùy runtime)

Folder ephemeral cho browser profile, file tạm, và debug captures.
Nó có thể chưa tồn tại nếu hệ thống chưa chạy runtime nào.

## Cách hiểu folder taxonomy

### Folder có dấu chấm ở root

Thường là namespace bền vững của pipeline hoặc control plane.

### Folder không có dấu chấm ở root

Thường là runtime source, docs source, hoặc công cụ phụ trợ.

## Folder nào quan trọng nhất để review hằng ngày

### Nếu review discovery

Nhìn vào `.narrative/`

### Nếu review quality của concept

Nhìn vào `.concepts/`

### Nếu review chiến dịch đã launch

Nhìn vào `.campaigns/`
