# Workspace Map

## Core folders

- `.agents/` -> control plane
- `.narrative/` -> Stage 1 outputs
- `.concepts/` -> Stage 2 outputs
- `.campaigns/` -> Stage 3 to Stage 6 outputs
- `.agents/evals/` -> Stage 7 learning packs
- `scripts/` -> runtime entrypoints
- `prompts/` -> prompt sources
- `src/` -> Control Room app
- `public/` -> generated mirror state and static assets
- `references/` -> reference-only material such as Paperclip

## Generated mirror

The app reads `public/control-room-state.json`.

Rebuild it with:

```powershell
npm run ops:sync
```
