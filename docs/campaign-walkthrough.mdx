# Campaign Walkthrough

Each campaign should feel like one tracked object, not a chain of forgotten
folders.

## Lifecycle

```mermaid
flowchart LR
    A["Trend Discovery"] --> B["Concept Lab"]
    B --> C["Content Creator"]
    C --> D["Launch Ops"]
    D --> E["Archive"]
    D --> F["AI MEME FACTORY"]
    F --> E
    E --> G["System Learning"]
```

## What the Control Room should show

- current stage
- current owner
- blockers
- next action
- trust links
- artifact refs

## What a good campaign looks like

- one clear winner
- clean stage contracts
- no silent handoffs
- archive and learning both completed
