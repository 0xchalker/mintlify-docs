# Introduction

MEME LABS is a campaign-first operating system for meme launches.

It does not start from "what coin should we make today?"

It starts from:

- what narrative is actually alive
- what proof exists
- what story is strong enough to deserve a full campaign

## The operating model

MEME LABS runs through seven stages:

1. Trend Discovery
2. Concept Lab
3. Content Creator
4. Launch Ops
5. Archive
6. AI MEME FACTORY
7. System Learning

The Control Room sits above those stages and makes the system manageable.

## What the Control Room does

- shows live campaigns
- shows current owner agents
- shows blockers and approvals
- shows run logs and trust links
- mirrors the filesystem into a readable ops surface

## What stays true

- `.narrative`, `.concepts`, `.campaigns`, and `.agents/evals` stay canonical
- browser X is the default public trust lane for Stage 1
- official X API is optional
- launch and onchain risk remain human-gated
