# Quickstart

## First commands

```powershell
npm install
npm run ops:sync
npm run dev
```

Open `/ops`.

## Default read order

1. [Control Room](/docs/operations/control-room)
2. [Architecture](/docs/architecture)
3. [Pipeline Overview](/docs/stages/overview)
4. [Campaign Walkthrough](/docs/campaign-walkthrough)

## Default operating rule

If the user is vague:

1. inspect the Control Room
2. route into the active campaign if one exists
3. otherwise start from Stage 1

## Stage 1 commands

```powershell
npm run grok:prepare
npm run trend:run
```

If a parsed batch already exists:

```powershell
npm run trend:review -- --input .narrative\...\raw-01
```
