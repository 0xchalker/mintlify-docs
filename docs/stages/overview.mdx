# Pipeline Overview

MEME LABS is a stage system wrapped by a campaign-first Control Room.

## Stage map

| Stage | Purpose | Key output |
| --- | --- | --- |
| Stage 1 | build a truthful narrative batch | `trend-final.md` + `trend-package/` |
| Stage 2 | lock one coin concept | `.concepts/[TICKER]/concept.md` |
| Stage 3 | build the public content system | `.campaigns/[TICKER]/content-system/` |
| Stage 4 | prepare and confirm launch | `.campaigns/[TICKER]/launch/` |
| Stage 5 | archive the campaign | `.campaigns/[TICKER]/` |
| Stage 6 | run the public loop | `.campaigns/[TICKER]/ai-meme-factory/` |
| Stage 7 | improve the system | `.agents/evals/...` |

## Control Room rule

Stages still own execution.
The Control Room owns visibility, routing, blockers, and next-step clarity.
