# Trend Discovery

## Goal

Build a narrative batch strong enough for Stage 2 to choose a winner.

## Decision owner

Discovery CEO

## Core outputs

- `review-scope.md`
- `batch-audit.md`
- `trend-final.md`
- `trend-package/`
- optional `x-public-log.*`

## Public trust layer

Stage 1 can narrate itself publicly on X, but that thread is never the real
handoff package.

Browser X is the default public transport for Stage 1.
Official X API is optional.

## Control Room impact

When Stage 1 finishes, the runtime should refresh the Control Room mirror so
the new winner, blockers, and trust links appear on `/ops`.
