# Runtime Commands

## Control Room

```powershell
npm run ops:sync
npm run dev
```

## Stage 1

```powershell
npm run grok:prepare
npm run trend:run
npm run trend:review -- --input <parsed-batch-dir>
```

## X API

```powershell
npm run x:api:doctor
npm run x:api:post -- --text "..."
npm run x:api:reply -- --text "..." --in-reply-to "<tweet-url-or-id>"
npm run x:api:thread -- --input "thread.json"
```

## Rule

Browser X is still the default Stage 1 public transport because it does not
consume API credits.
