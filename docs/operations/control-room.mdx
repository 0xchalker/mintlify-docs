# Control Room

The Control Room is the operator surface for MEME LABS.

Routes:

- `/ops`
- `/ops/campaigns`
- `/ops/campaigns/$campaignId`
- `/ops/agents`
- `/ops/runs/$runId`
- `/ops/inbox`

## What it shows

- campaign board
- stage owner
- blockers
- next action
- approvals
- run timeline
- trust links

## What it does not replace

The Control Room does not replace the canonical artifact tree.

It mirrors:

- `.narrative/`
- `.concepts/`
- `.campaigns/`
- `.agents/evals/`
