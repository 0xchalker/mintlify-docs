# MEME LABS Docs

This repository contains the Mintlify docs portal for MEME LABS.

Core entrypoints:

- `docs.json`
- `index.mdx`
- `docs/`

The docs are organized around the MEME LABS operating system:

- Step 1: Trend Discovery
- Step 2: Concept Lab
- Step 3: Content Creator
- Step 4: Launch Ops
- Step 5: Archive
- Step 6: AI MEME FACTORY
- Step 7: System Learning

It also documents:

- output contracts for `.narrative/`, `.concepts/`, `.campaigns/`, and `.agents/evals/`
- automation and orchestration for Step 1 and AI MEME FACTORY
- the full campaign walkthrough for end-to-end review
